## 时延检查记录

| 链路 | 检查点 | 状态 | 备注 |
|------|--------|------|------|
| 截图 | trigger_capture → 覆盖层显示 | | 应用侧延迟 < 200ms，主要等待屏幕捕获 |
| 问答 | 提交 → 首字流式输出 | | 依赖 API 延迟，流式首字约 1-3s |
| 搜索 | 提交 → 搜索完成 → 首字输出 | | Tavily + LLM 双延迟，约 3-8s |
| 存储 | append_turn 写入 | | 本地 JSON 写入，< 50ms |

## 备注

- 所有测试在 Windows 10/11 上执行
- LLM Provider 需配置有效 API Key 才能验证问答流程
- Tavily API Key 为可选项，不配置时仅测试直答模式
